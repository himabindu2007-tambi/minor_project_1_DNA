# 🧬 GroupDNA – WhatsApp Group Chat Analyzer

> **Unlock the DNA of your group conversations with Python!**

GroupDNA is a Python-based data analytics project that transforms raw WhatsApp chat exports into meaningful insights. It analyzes group conversations to uncover activity patterns, user engagement, communication habits, and personality archetypes.
## 📌 Project Overview

GroupDNA reads exported WhatsApp chat data and generates a detailed analytics report containing:
* 📊 Group Overview
* 📅 Most Active Day
* ⏰ Most Active Hour
* 🔥 Activity Heatmap
* 💬 Top Used Words
* ⚡ Response Time Analysis
* 🤫 Silent Streak Detection
* 🎭 Personality Archetype Detection

The project demonstrates how Python can be used to analyze unstructured text data without relying on advanced data analysis libraries.

## ✨ Features

### 📂 Chat Parser

* Reads exported WhatsApp chat files
* Parses timestamps, users, and messages
* Handles media and deleted messages

### 📈 Group Overview

* Total Messages
* Total Participants
* Chat Duration
* Message Distribution

### 📅 Activity Analysis

* Busiest Day
* Busiest Hour
* User Activity Statistics

### 🔥 Activity Heatmap

* Hour-wise activity visualization using NumPy
* Easy-to-read heatmap representation

### 💬 Word Analysis

* Most Frequently Used Words
* Communication Trends

### ⚡ Response Analysis

* Fastest Replier
* Slowest Replier
* Silent Streak Detection

### 🎭 Personality Detection

Automatically classifies users into fun personality categories such as:

* 🚀 The Spammer
* ❤️ The Group Mom
* 🌙 The Night Owl
* 📖 The Storyteller
* 🎭 The Drama Queen
* 👻 The Ghost
